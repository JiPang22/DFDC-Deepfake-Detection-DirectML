# DFDC-Deepfake-Detection-DirectML

##Deepfake detection system using ResNet18, optimized for AMD GPUs (DirectML) on WSL2.

Model: ResNet18 (Binary Classification)

Hardware: AMD Radeon RX 6600 (DirectML)

Dataset: Facebook DFDC (Deepfake Detection Challenge)

Accuracy: Train 97.43% / Val 85.68% (Chapter 7 기준)
